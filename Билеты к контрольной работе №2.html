1. Определение устойчивости по Ляпунову

Пусть есть система x=f(t,x). Решение x=ϕ(t) этой системы называется устойчивым по Ляпунову, если для ∀ϵ>0∃δ(ϵ)>0:∀x0 выполняется |ϕ(t0)−x0|<δ решение x=ψ(t) системы x=f(t,x) с начальным условием ψ(t0)=x0 определено при t≥t0.

2. Определение ассимптотической устойчивости по Ляпунову

Если кроме устойчивости по Ляпунову выполняется limt→∞|ψ(t)−ϕ(t)|=0, то решение x=ϕ(t) называется ассимптотически устойчивым.

3. Теорема об ассимптотической устойчивости нулевого положения равновесия

Для того чтобы положение равновесия x=0 системы x=f(t,x) было ассимптотически устойчивым, необходимо и достаточно, чтобы все собественные значения матрицы A имели отрицательные действительные части.

6. Теорема об ассимптотической устойчивости и неустойчивости нормальной системы (2 Теорема Ляпунова)

Пусть имеется нормальная система x=Ax+F(t,x), где A - постоянная матрица. Пусть также при всех t≥t0 и достаточно малом |x| выполняется |F(t,x)|≤M|x|1+αα,M>0 Тогда положение равновесия x=0 системы x=Ax+F(t,x):

  1. Ассимптотически устойчиво по Ляпунову, если все собственные значения матрицы A имеют отрицательные действительные части.
  2. Неустойчиво, если среди собственных значений матрицы A имеется хотя бы одно с положительной действительной частью.
